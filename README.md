# AI for Price Prediction

## Overview
This project predicts commodity prices using historical data, weather conditions, and market trends. It includes:
- ARIMA, XGBoost, and LSTM models.
- A Streamlit dashboard for visualization.
- Hyperparameter tuning and feature importance analysis.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`.
2. Preprocess data: `python preprocess.py`.
3. Train models: `python train.py`.
4. Run the dashboard: `streamlit run dashboard.py`.
